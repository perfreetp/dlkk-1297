# 独立开发者收入复盘平台 - 产品需求文档

## 1. 产品概述

**复盘台 (RevueDesk)** - 专为独立开发者和小型团队打造的收入管理与协作平台。通过系统化的收入追踪、实验记录和团队复盘，帮助用户洞察产品经营状况，优化变现策略，实现数据驱动的业务增长。

- 目标用户：独立开发者、SaaS 创业者、小型产品团队
- 核心价值：一站式记录产品经营数据、追踪实验效果、沉淀团队洞察
- 市场定位：聚焦"轻量级 + 强协作"的收入复盘工具

## 2. 核心功能模块

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 团队管理员 | 邮箱注册 | 创建小组、管理成员、设置提醒、全功能访问 |
| 团队成员 | 受邀加入 | 参与讨论、编辑实验记录、查看小组内容 |
| 访客 | 无需注册 | 仅查看公开示例数据 |

### 2.2 页面结构

1. **仪表盘 (Dashboard)** - 数据总览与关键指标
2. **收入流水 (Revenue)** - 多渠道收入追踪与趋势分析
3. **实验记录 (Experiments)** - 产品实验与 A/B 测试追踪
4. **复盘文章 (Reviews)** - 经营笔记与复盘提纲生成
5. **同行小组 (Groups)** - 团队协作与知识沉淀

## 3. 页面详细功能

### 3.1 仪表盘页面

**模块组成：**
- **月度收入概览卡片** - 显示当月总收入、各渠道占比、同比增长率
- **收入趋势图** - 近 6 个月收入折线图（支持按产品筛选）
- **实验效果排行** - 展示近期实验的收益变化 TOP 5
- **待办提醒面板** - 下次复盘时间、待验证想法数量、待点评笔记
- **快速操作入口** - 快捷添加收入记录、创建实验、发布笔记

**数据来源：** 收入流水表、实验记录表、复盘文章表

### 3.2 收入流水页面

**收入类型支持：**
- **订阅收入** - 月订阅、年订阅，按周期自动统计
- **一次性购买** - 独立产品销售，包含首次购买和升级
- **广告收入** - 展示广告、激励视频、赞助内容

**功能特性：**
- **收入录入** - 表单支持：金额、日期、产品、渠道、类型、备注
- **产品筛选** - 多选产品标签过滤
- **月度趋势图** - 堆叠面积图展示各收入类型占比变化
- **渠道对比** - 柱状图对比不同渠道收益
- **导出功能** - 支持导出 CSV 格式月度报表

**数据字段：**
- 金额、币种、日期
- 产品名称、产品标签
- 收入类型（订阅/一次性/广告）
- 渠道来源（App Store/Google Play/官网/Direct）
- 关联实验（可选）
- 备注说明

### 3.3 实验记录页面

**实验类型：**
- **价格调整实验** - 记录定价变更前后收入对比
- **渠道投放实验** - 不同渠道的投入产出比
- **功能实验** - 特性上线对转化率的影响

**实验记录字段：**
- 实验名称、描述、类型
- 目标指标（如：提升月收入 20%）
- 假设前提
- 开始/结束日期
- 实验组/对照组设置
- 结果指标（实际达成情况）
- 结论总结
- 相关收入记录关联

**功能特性：**
- **实验看板** - 卡片式展示所有实验，支持状态筛选（进行中/已完成）
- **目标追踪** - 进度条可视化展示目标达成率
- **结果对比** - 前后数据对照视图
- **自动提纲生成** - 基于实验数据生成复盘提纲

### 3.4 复盘文章页面

**文章类型：**
- **经营笔记** - 日常经营心得，仅小组可见
- **复盘提纲** - 由实验数据或手动创建的结构化复盘模板
- **可借鉴做法** - 标记为"值得参考"的团队知识

**功能特性：**
- **富文本编辑** - Markdown 支持，实时预览
- **提纲生成** - 基于选定实验自动生成复盘结构
- **可见性控制** - 私密/仅小组成员可见
- **点评功能** - 成员可对笔记进行评论和点赞
- **收藏功能** - 收藏其他成员的可借鉴做法
- **标记待验证** - 将想法标记为待验证假设

**文章结构：**
- 标题、标签、创建日期
- 作者信息
- 内容（Markdown）
- 可见性设置
- 关联实验（可选）
- 点评列表
- 收藏数、阅读数

### 3.5 同行小组页面

**小组管理：**
- 创建小组、邀请成员（邮箱/链接）
- 成员角色设置（管理员/成员）
- 小组设置（名称、描述、图标）

**协作功能：**
- **经营笔记发布** - 发布仅小组成员可见的笔记
- **成员点评** - 对笔记进行评论和反馈
- **可借鉴做法收藏** - 收藏值得参考的做法
- **待验证想法标记** - 记录待验证的商业假设
- **复盘提醒设置** - 设置下次复盘日期，自动提醒

**知识沉淀：**
- 收藏夹 - 个人收藏的所有可借鉴做法
- 想法池 - 标记待验证的想法列表
- 提醒日历 - 下次复盘时间线

## 4. 核心业务流程

### 4.1 收入记录流程

```
用户进入收入流水页面
  ↓
点击"添加收入"按钮
  ↓
填写收入表单（金额、产品、类型、渠道等）
  ↓
保存成功 → 更新仪表盘数据 → 生成趋势图表
```

### 4.2 实验追踪流程

```
用户创建实验记录
  ↓
设置实验目标与假设
  ↓
关联相关收入记录
  ↓
实验进行中 → 更新进度
  ↓
实验结束 → 记录结果 → 自动生成复盘提纲
  ↓
发布复盘文章 → 小组成员点评
```

### 4.3 团队协作流程

```
管理员创建小组
  ↓
邀请团队成员加入
  ↓
成员发布经营笔记
  ↓
其他成员点评互动
  ↓
设置下次复盘提醒
  ↓
到期提醒 → 开展复盘会议
```

## 5. 用户界面设计

### 5.1 设计风格

**视觉定位：数据仪表盘风格 + 极简主义**

- **主色调**：深青蓝色 (#1a365d) - 专业可信
- **强调色**：活力橙 (#ed8936) - 行动号召、数据高亮
- **辅助色**：
  - 成功绿 (#38a169) - 正向指标
  - 警示红 (#e53e3e) - 负向变化
  - 中性灰 (#718096) - 次要信息

**排版系统：**
- 主字体：Noto Sans SC（中文）+ Inter（英文/数字）
- 标题层级：H1: 32px / H2: 24px / H3: 18px
- 正文：15px，行高 1.6
- 数据展示：Roboto Mono（等宽数字）

**组件风格：**
- 卡片式布局，16px 圆角
- 阴影层次：sm（轻柔）/ md（悬浮）/ lg（弹窗）
- 按钮：8px 圆角，主按钮使用渐变背景
- 输入框：4px 圆角，2px 边框焦点高亮

**图标风格：**
- 使用 Lucide Icons（线性风格，2px 描边）
- 保持一致的 20px 尺寸

### 5.2 页面布局

**整体结构：**
```
┌─────────────────────────────────────────────┐
│  顶部导航栏（Logo + 导航菜单 + 用户信息）     │
├─────────────────────────────────────────────┤
│                                             │
│              主内容区域                       │
│                                             │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐     │
│  │  卡片1  │  │  卡片2  │  │  卡片3  │     │
│  └─────────┘  └─────────┘  └─────────┘     │
│                                             │
│  ┌─────────────────────────────────────┐   │
│  │           主数据区域                  │   │
│  │                                     │   │
│  └─────────────────────────────────────┘   │
│                                             │
└─────────────────────────────────────────────┘
```

**Dashboard 布局：**
- 顶部：关键指标卡片行（4 个等宽卡片）
- 中部：双列布局（左侧趋势图，右侧实验排行）
- 底部：待办提醒面板

**Revenue 布局：**
- 顶部：筛选工具栏 + 添加按钮
- 左侧：月度趋势堆叠面积图（60% 宽度）
- 右侧：收入类型饼图（40% 宽度）
- 底部：收入流水表格（支持分页）

**Experiments 布局：**
- 顶部：实验类型标签筛选 + 创建按钮
- 主区域：Kanban 看板式布局（待开始/进行中/已完成）
- 每个实验卡片：名称、进度条、目标、状态

**Reviews 布局：**
- 顶部：类型筛选标签（笔记/提纲/可借鉴）
- 主区域：卡片网格布局
- 侧边栏：我的收藏、待验证想法

**Groups 布局：**
- 左侧：小组列表 + 成员管理
- 主区域：内容展示区（笔记列表/提醒日历）
- 侧边栏：快捷操作面板

### 5.3 响应式设计

**断点设置：**
- 桌面端：≥ 1280px（完整布局）
- 平板端：768px - 1279px（双列转单列）
- 移动端：< 768px（抽屉式导航，卡片堆叠）

**移动端适配：**
- 底部 Tab 导航替代侧边栏
- 卡片全宽显示
- 图表简化为关键数据展示
- 抽屉式筛选面板

### 5.4 动效与交互

**页面切换：**
- 内容淡入 + 上移 20px，300ms ease-out
- 标签页切换使用滑块指示器

**数据更新：**
- 数字变化使用计数动画（1s duration）
- 图表重绘使用过渡动画

**悬停效果：**
- 卡片悬停：上移 4px + 阴影加深
- 按钮悬停：背景色加深 10%
- 表格行悬停：背景色变为浅灰

**反馈提示：**
- 操作成功：顶部滑入绿色 Toast，3s 自动消失
- 表单错误：输入框边框变红 + 下方错误提示
- 删除确认：Modal 弹窗确认
